# FUTURE_DS_01
Sales Analysis - Task 1
This repository contains my submission for Task 1 of the Future Interns Data Science & Analytics internship.

In this project, I am analyzing a sales dataset to understand revenue trends, identify top-performing products, and evaluate regional performance. The goal is to extract meaningful insights that can help in making better business decisions.

The analysis includes data cleaning, exploration, and visualization using tools like Power BI / Excel. A dashboard and detailed insights will be added as the project progresses.
